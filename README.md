# 時間割アルファ

履修登録を手助けするiOSアプリ。登録モードと参照モードに分かれ、直感的な操作で時間割を管理できます。

## ユーザー要件

1. **履修登録モード**: 授業の追加・削除・移動が可能
2. **参照モード**: 閲覧専用だがメモは編集可能
3. **同一コマ複数授業**: 科目ごとに優先順位をつけて比較検討可能
4. **履修確定ボタン**: 被りがあれば警告を表示
5. **ロック機能**: 選択した授業をそのコマから外せなくする
6. **授業属性**: 単位数、指導教員、メモ（画像・PDF添付可）
7. **画面下半分参照**: 登録モードでPDF/Web/画像を参照可能
8. **画像共有**: 時間割を画像で共有
9. **OCR読み取り**: 画像から授業名を文字起こしして候補に追加
10. **授業カラー**: コンテナの色を自由に変更、一括変更も可能
11. **サブタイトル**: 授業のサブタイトルを任意で追加可能
12. **汎用メモ**: 科目に関係ないメモ帳（iOS純正メモ風）
13. **スマホ操作性**: タッチターゲット44px以上、スクロール不要で全体表示

## 技術スタック

- **言語**: Swift 6.0
- **UI**: SwiftUI
- **データ**: SwiftData
- **対応OS**: iOS 17.0+
- **フレームワーク**: PDFKit, WebKit, Vision, PhotosUI, UserNotifications, AVFoundation

## 変更履歴

### 2025-04-01: UI/UX改善（Emil Kowalski哲学 + pbakaus/impeccable準拠）

**変更ファイル**:
- `TimetableView.swift` — ヘッダーUI全面改善
- `TimetableCellView.swift` — タップ操作・タッチターゲット改善
- `CourseDetailView.swift` — 参照モードでもメモ編集可能に

**改善内容**:

| 症状 | 原因 | 解決策 |
|------|------|--------|
| 確定ボタンと操作が曖昧 | 1行ヘッダーに全操作を詰め込んでいた | 2段ヘッダー: 上段=学期名+モード切替、下段=モード別アクションバー |
| モードの状態が分かりにくい | アイコンのみの切替ボタン | Capsule型セグメントコントロール（登録=青、参照=グレー） |
| デッドスペースがある | ヘッダー高さ見積もりが不正確 | compactHeader高さを76pxに修正 |
| スマホで操作しにくい | タッチターゲットが小さい、タップで詳細開けない | セルタップで詳細画面遷移、ボタンパディング拡大 |
| 参照モードでメモ編集不可 | isEditable がモード連動していた | メモのisEditableを常にtrueに変更 |
| 確定後のフローが不明確 | アラートが簡素 | 確定後に自動で参照モードに切替+成功ハプティック |
| モード切替が唐突 | トランジションなし | asymmetricトランジション（登録バー:slide+fade、参照バー:scale+fade） |
